ITEIS App

IT Equipment Inventory System is simple app made for SDAxCodecool's recruitment competition.

In order to use the app, first you need to register (as a default user)

Since you're logged in, use some links from navbar.

Admin can see all users list, can change employees details and add devices for specific employee

Unfortunately, front is made in thymeleaf, I didn't have time to learn React in like 7 days. Nonetheless I know that we have to use axios to consume RestAPI in front-end app.
